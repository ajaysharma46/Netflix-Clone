# 🎬 Netflix Clone (100% Responsive)

A fully responsive **Netflix Clone** built using modern web technologies. This project replicates the core UI and user experience of Netflix, ensuring seamless performance across all devices — from mobile phones to large desktop screens.

---

## 🚀 Live Demo

👉 https://ajay-netflix-clone.vercel.app/


---

## 📌 Features

* 📱 **100% Responsive Design**
  Optimized for mobile, tablet, and desktop screens.

* 🎨 **Modern UI/UX**
  Clean and visually appealing interface inspired by Netflix.

* 🔍 **Dynamic Content Sections**
  Organized movie/show rows similar to Netflix layout.

* ⚡ **Fast Performance**
  Lightweight and optimized for smooth user experience.

* 🎥 **Interactive Elements**
  Hover effects, banners, and engaging UI components.

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript** – Interactivity

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/ajaysharma46/netflix-clone.git
```

2. Navigate to the project folder:

```bash
cd netflix-clone
```

3. Open `index.html` in your browser.
